# Servicios AWS

### Elastic Beanstalk
Permite a los clientes migrar, implementar y escalar fácilmente aplicaciones full stack. Gestiona las operaciones de infraestructura y aplicaciones, lo que permite a los desarrolladores centrarse en la lógica empresarial.

### Control tower
Organiza las capacidades de varios otros AWS servicios, como AWS Organizations AWS Service Catalog, y AWS IAM Identity Center, crear una landing zone en menos de una hora

### AWS Budgets
Establece un presupuesto mensual de gastos con un importe objetivo fijo para realizar un seguimiento de todos los gastos asociados a tu cuenta.
